Sherlock is a browser-based AI workspace for investigations, research boards, grounded chat, timeline analysis, network graphs, and live monitoring. You bring your own provider key, and Sherlock keeps workspace data browser-local.

Use this page as the front door for the docs. Start with the quick setup path, then read the concepts that match the work you are doing.
